Words - your new learning solution

Studying for a test can be difficult, therefore you can now use words: a completely free solution for learning online. Randomize the order of questions, practice in flashcard mode or use another form of learning: it's up to you.


To start learning you have to first create a list using the list creator tool. Once you're satisfied with that you can download the list by clicking the download button, which will generate a JSON file with the list in it.


Now for the testing: go to the testing page and upload the JSON file you just downloaded. Then select your options (e.g. whether the list should be asked in random order) and then select your test type. You're all set! Happy learning :)
